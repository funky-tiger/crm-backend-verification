# description

- 一个基于 ant-design-pro 的后端权限验证的 CRM 系统

## 如何使用？

1. 安装 npm

```bash
npm install
```

2. 启动本地服务器

```bash
npm start
```

3. build 项目

```bash
npm run build
```
